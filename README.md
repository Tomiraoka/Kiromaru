## 📚 About the Project
Kiromaru is a multi-page anime and movie discovery platform. It showcases core frontend skills: semantic HTML, responsive CSS (Grid & Flexbox), and interactive JavaScript including DOM manipulation, event handling, dynamic data rendering, and search filtering.

## 🧩 Pages
- Home – Welcome page with an interactive hero slider and top recommendations
- Anime – Browse anime by categories like Popular, Isekai, Romance, and Comedy
- Movies – Discover top-rated and trending anime films with visually immersive backgrounds
- News – Read the latest updates and announcements in the anime world
- Details Pages – Dedicated pages for specific anime and films with synopses, ratings, and embedded video players

## 🔍 Interactive Features
- Real-time Search – Dynamic search bar with a dropdown displaying matching titles and images instantly
- Interactive Carousels – Smooth scrolling sliders for browsing different content categories
- Dynamic Content Loading – Populating details pages automatically based on the user's selection

## ⚙️ JavaScript Features
- Variables & Data Types (objects and arrays for storing anime/film data)
- Conditionals & Loops (filtering search results, rendering carousels)
- Custom Functions (opening content details, scrolling sliders, handling search logic)
- Event Handling (click events, keyboard inputs for search functionality)
- DOM Manipulation (updating page content, toggling classes, dynamic styling)
- URL Parameters (parsing query strings to load specific data on details pages)
